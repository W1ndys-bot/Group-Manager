# GroupManager

## 命令

ban@ 时间 禁言 x 秒，默认 60 秒

unban@ 解除禁言

banme 随机禁言自己随机秒

banrandom 随机禁言一个群友随机秒

banall 全员禁言

unbanall 全员解禁

t@ 踢出指定用户

recall 撤回消息(需要回复要撤回的消息)

vc-on 开启视频监控

vc-off 关闭视频监控

wf-on 开启欢迎欢送

wf-off 关闭欢迎欢送
